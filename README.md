# GitHub Copilot Training Labs

A comprehensive collection of hands-on labs designed to help you master GitHub Copilot for various development scenarios including AI-driven coding, API development, testing, security auditing, CI/CD pipelines, and documentation.

## 📚 Labs Overview

### 1. [Building Web Services with AI](1.Building%20Web%20Services%20with%20AI.md)
Integrate Copilot into VS Code to build and adapt Python web services using Flask and FastAPI. Learn to create REST APIs and generate tests without leaving your editor.

### 2. [Update a Web API with GitHub Copilot](2.Update%20a%20Web%20API%20with%20GitHub%20Copilot.md)
Learn to modify existing Python web applications using advanced Copilot techniques. Add new API endpoints and create tests for specific routes.

### 3. [Generate Unit Tests](3.%20Generate%20unit%20tests.md)
Accelerate unit test development using GitHub Copilot. Generate test cases, methods, assertions, and mocks for your applications.

### 4. [Unit Test Python](4.unit%20test%20python.md)
Comprehensive lab for building Python applications with automated testing. Create a Calculator class, generate test suites, practice Test-Driven Development (TDD), and generate GitHub Actions pipelines.

### 5. [Copilot CLI](5.%20Copilot%20CLI.md)
Learn to use the GitHub Copilot CLI in Codespaces for interactive development, code generation, and system administration tasks.

### 6. [The Boilerplate Buster](6.The%20Boilerplate%20Buster.md)
Spin up a GitHub Codespace to rapidly generate a complete project skeleton for a Dockerized Python web server using FastAPI with proper testing and deployment configurations.

### [cicd pipeline](cicd%20pipeline.md)
Automate your testing workflow using GitHub Actions. Design and deploy a CI/CD pipeline that runs tests on every commit, includes linting, and uploads code coverage reports.

### [Documentation & Platform Alignment](Documentation%20&%20Platform%20Alignment.md)
Master documentation and communication in professional environments. Generate production-ready README files, track code changes with Git diffs, and create structured Pull Request descriptions.

### [The Silent Vulnerability](The%20Silent%20Vulnerability.md)
Act as an Application Security Engineer to audit for vulnerabilities, prove exploits with test cases, and use Agent Mode to securely refactor code to fix SQL injection vulnerabilities.

---

## 🛠️ Prerequisites

- GitHub account with Copilot subscription
- Visual Studio Code (optional, depending on lab)
- Python 3 (for labs involving Python development)
- Basic understanding of programming concepts

## 📖 Getting Started

1. Clone or navigate to this repository
2. Choose a lab based on your learning goals
3. Follow the instructions in each lab's markdown file
4. Most labs require GitHub Codespaces - the repository links are provided in individual lab files

## 📝 Notes

- Some labs reference external repositories for pre-configured environments
- Always clean up your Codespaces after completing labs to conserve your free monthly compute quota
- Refer to individual lab files for specific prerequisites and setup instructions
